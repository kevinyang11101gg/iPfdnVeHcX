# Python协同过滤小说推荐系统

## 核心功能
- 用户登录注册
- 小说信息推荐
- 查阅小说详情
- 小说数据可视化大屏
- 公告信息查看
- 个人中心管理
- 用户管理
- 小说信息管理
- 系统管理

## 适用场景
- 在线小说阅读平台
- 文学爱好者的个性化推荐服务
- 小说网站内容优化与用户互动

## 技术特点
- 使用Django框架构建后端
- 基于MySQL数据库存储用户和小说数据
- 前端界面采用Vue技术
- 支持数据可视化大屏展示
- 系统架构稳定，运行稳定
